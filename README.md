# Python AST Code Analyzer

A simple static code analysis tool built using Python's AST module.
It parses Python source files to extract functions, classes, imports,
and their line numbers, and outputs a structured JSON summary.

## Features
- AST-based static analysis (no code execution)
- Extracts functions, classes, imports
- Includes line numbers for definitions
- Outputs JSON

## How to Run
```bash
python my_ast_analyzer.py
